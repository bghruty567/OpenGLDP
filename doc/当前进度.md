# 当前进度

更新时间：`2026-04-17`

## 1. 已完成

当前梯度模块已经完成以下收敛：

1. 规则网格公开主线固定为 `FD`。
2. 非结构网格公开主线固定为统一 `AWLS`。
3. GUI 已不再保留旧的 `WeightedLeastSquares` 选项。
4. benchmark 解析数组默认关闭，只在测试程序显式开启。
5. 已删除旧的稀疏梯度相关 shader：
   - `RGGradient.glsl`
   - `USGGradient.glsl`
   - `SparseGradient.glsl`
   - `SparseReconstruct.glsl`

## 2. 当前对外可说清楚的方案

当前可以非常明确地说：

- `vtkImageData / vtkRectilinearGrid / vtkStructuredGrid` -> `FD`
- `vtkUnstructuredGrid` -> `AWLS`

对于 `vtkUnstructuredGrid` 内部各种单元类型，现在不再按单元类型切很多默认算法分支。

## 3. 当前验证结论的推荐口径

如果以你这轮最新测试为准，而不是 `results/` 旧文件：

- 解析场测试是当前主要正确性依据；
- 真实数据相对 VTK 的困难样例偏差应写成局限；
- 这版可以作为本科毕设的最终工程方案，但不能声称已全面等价于 VTK。

## 4. 还建议继续做的事

1. 重新跑一轮 benchmark 并更新 `results/`。
2. 把解析场、真实数据、参数消融、性能统计四类实验都整理进论文。
3. 在论文中单独讨论 `ShipHull_0` 和 `notch_stress`。
