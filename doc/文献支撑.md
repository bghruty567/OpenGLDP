# 文献支撑

## 1. 文档目的

本文档用于回答两个问题：

1. 当前项目的理论基础到底参考了哪些文献；
2. 论文、答辩或技术说明中，哪些说法是有文献依据的。

## 2. 当前方案与文献的关系

当前项目不是“逐行复刻某一个开源实现”，而是：

- 以非结构网格最小二乘梯度重构文献为理论基础；
- 借鉴 OpenFOAM 的工程化 least-squares 路线；
- 参考 VTK 的梯度接口与输出约定作为对照基线；
- 在本项目的数据结构和 OpenGL 执行框架下进行统一实现。

更准确的表述是：

> 本项目在非结构网格上采用基于局部 stencil 的加权最小二乘梯度重构，并结合邻域扩展、局部维数识别和正则稳定化完成工程实现；在验证上，优先采用解析梯度与网格收敛思路，再用 `vtkGradientFilter` 做工程基线比较。

## 3. 关键文献与支持点

## 3.1 Mavriplis & Thomas, 2003

文献信息：

- Dimitri J. Mavriplis
- James L. Thomas
- *Revisiting the Least-squares Procedure for Gradient Reconstruction on Unstructured Meshes*
- NASA/CR-2003-212683

这篇文献主要支持以下论点：

1. 非结构网格梯度重构可采用 least-squares 路线；
2. 线性精确性是梯度重构的基本要求；
3. 权重与 stencil 设计会明显影响精度；
4. 在高拉伸或退化网格上，最小二乘方法可能出现明显精度退化。

这与本项目的直接对应关系是：

- `AdaptiveWLS.glsl` 的核心仍是局部最小二乘求解；
- `wlsExponent`、`wlsLambda` 对应距离权重和正则化；
- `ensureAdaptiveSupport(...)` 中的邻域扩展和质量评估，就是针对 stencil 敏感性做的工程增强。

## 3.2 Wang et al., 2019

文献信息：

- Nianhua Wang
- Ming Li
- Rong Ma, et al.
- *Accuracy analysis of gradient reconstruction on isotropic unstructured meshes and its effects on inviscid flow simulation*
- *Advances in Aerodynamics*, 2019

这篇文献主要支持以下论点：

1. least-squares 梯度在测试的非结构网格上至少具有一阶精度；
2. 梯度重构的验证应看解析函数、误差范数和网格收敛；
3. gradient reconstruction 的精度会直接影响后续数值解的精度；
4. 可以结合 MMS 做更系统的精度研究。

这与本项目的直接对应关系是：

- 新版 `TestGradient` 把 `*_exact_grad` 作为最高优先级参考；
- 新增多组标量/向量 benchmark 以支持解析场验证；
- 输出 `NMAE_vec`、`NRMSE_vec`、角度误差等指标，便于批量比较。

## 3.3 OpenFOAM least-squares 文档

OpenFOAM 文档明确说明其 least-squares 梯度是“cell gradient is calculated using least squares”，并提供了：

- `leastSquares`
- `pointCellsLeastSquares`
- `edgeCellsLeastSquares`

这说明：

1. 基于局部 stencil 的 least-squares 梯度是成熟工程路线；
2. stencil 选择本身就是方案设计的一部分；
3. 工程实现中可以针对不同连接关系选择不同邻域。

本项目参考的是这种“stencil + least-squares”的思路，但做了统一扩展：

- 同时支持点数据和单元数据；
- 同时支持解析 benchmark、VTK 参考和 CSV 批量统计；
- 通过 OpenGL 而不是 CPU 循环求解局部重构。

## 3.4 VTK `vtkGradientFilter`

VTK 文档说明 `vtkGradientFilter` 是一个通用梯度估计过滤器，并指出：

- 具体梯度计算依赖数据集类型；
- 边界不是中心差分；
- 输出分量数是输入分量数的 3 倍；
- 对向量场输出顺序为 `{du/dx, du/dy, du/dz, ...}`。

这支持当前项目中的两点做法：

1. 把 VTK 当作工程基线，而不是数学真值；
2. 在测试程序中使用与 VTK 相同的展开顺序，便于比较。

## 3.5 解析解 / 制造解验证文献

Oliver 等人的 2013 年工作采用了解析解和制造解相结合的方式，对三维非结构有限元方法进行正式验证，并显式报告误差与收敛率。这类文献支持以下说法：

1. 解析解和制造解是成熟的数值验证方法；
2. 只看某一组真实数据对照不足以证明算法正确；
3. “解析场 + 收敛 + 工程样例”是更可靠的测试组合。

## 4. 论文中建议使用的表述

## 4.1 关于方法来源

建议写法：

> 本文在非结构网格上采用基于局部支撑的加权最小二乘梯度重构方法。该方法以最小二乘梯度重构文献为理论基础，并结合本系统的数据结构、GPU 并行执行方式和稳定性需求进行了工程化改造。

不建议写成：

> 本文方法完全等价于 OpenFOAM / VTK / 某篇论文中的原始实现。

## 4.2 关于测试方法

建议写法：

> 在验证策略上，本文优先采用解析场 exact gradient 对梯度计算模块进行正确性验证，并结合网格收敛和误差范数分析评价算法精度；在缺少解析真值的真实数据上，再以 `vtkGradientFilter` 作为工程对照基线。

## 4.3 关于真实数据与 VTK 的差异

建议写法：

> 对复杂真实数据，本文方法与 `vtkGradientFilter` 的结果仍存在一定差异。这一现象主要反映了非结构网格局部质量、边界处理和 stencil 选择对梯度重构的影响，不应简单等同为算法失效。

不建议写成：

> 只要与 VTK 不一致，就说明当前算法还不能使用。

| 结论 | 文献支撑 |

| 验证应优先看解析梯度、误差范数和收敛性 | [2][5] |

## 6. 推荐在论文中保留的参考文献

若只保留最关键的 4 到 5 篇，建议如下：

1. Mavriplis & Thomas, 2003  
   用于说明最小二乘梯度重构理论、线性精确性、权重/stencil 敏感性。
2. Wang et al., 2019  
   用于说明梯度重构精度分析、解析函数验证、网格收敛和 MMS。
3. OpenFOAM least-squares 文档  
   用于说明 least-squares 在工程 CFD 中的成熟应用。
4. VTK `vtkGradientFilter` 文档  
   用于说明工程基线的来源和输出组织方式。
5. Oliver et al., 2013  
   用于说明解析解和制造解联合验证的合理性。
