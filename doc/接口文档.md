# OpenGLDP 系统类详细参考

## 1. DataObject.h

### 1.1 enum DataArrayType
- POINT_DATA: 点关联数组
- CELL_DATA: 单元关联数组

### 1.2 struct DataArray
- name: 数组名
- data: 扁平浮点数据
- numComponents: 每个元组分量数
- dataType: 点/单元关联类型

### 1.3 enum GridType
- DATA_OBJECT_TYPE_RegularGrid: 规则网格
- DATA_OBJECT_TYPE_UNSTRUCTURED: 非结构网格

### 1.4 class DataObject
#### 成员变量
- gridType: 网格类型
- points: 点坐标，格式[x0,y0,z0,...]
- cellCenters: 单元中心坐标
- dataArrays: 所有数据数组
- pointNeighbors / pointNeighborOffsets: 点邻接CSR
- pointInCellNeighbors / pointInCellNeighborOffsets: 点-单元CSR
- cells / cellTypes / cellOffsets: 单元连接关系与类型
- cellNeighbors / cellNeighborsOffsets: 单元邻接CSR
- dimensions[3]: 规则网格维度

#### 成员函数
- findDataArray(name,type): 查找数组（可变）
- findDataArray(name,type) const: 查找数组（只读）
- upsertDataArray(name,data,numComponents,type): 更新或插入数组
- pointCount(): 点数量
- cellCount(): 单元数量

## 2. VTKDataConverter.h/.cpp

### 职责
- vtkDataSet -> DataObject
- DataObject -> vtkDataSet（用于渲染与结果导出）

### 核心变量
- vtkData: 当前VTK数据对象
- internalData: 当前内部数据对象

### 核心函数
- bindVTKDataAndInternalData(vtkData, internalData): 绑定对象
- convertVTKToInternal(): 总入口（含类型分派）
- convertInternalToVTK(): 反向转换入口

### 子函数
- convertType(): 识别规则/非结构网格
- convertPoints(): 点坐标转换
- convertDataArrays(): 点/单元数组转换
- convertDimensions(): 规则网格维度
- convertCell(): 单元连接与类型
- convertCellCenters(): 单元中心
- convertPointInCellNeighbors(): 点所属单元邻域
- convertPointNeighbors(): 拓扑点邻域
- convertPointNeighborsByKNN(k): KNN点邻域
- convertPointNeighborsRobust(minK,knnK): 拓扑+KNN补全
- convertCellNeighbors(): 拓扑单元邻域
- convertCellNeighborsByKNN(k): KNN单元邻域
- convertRegularGrid(): 规则网格转换流程
- convertUnstructuredGrid(): 非结构网格转换流程

---

## 3. GLGradientEngine.h/.cpp

### 职责
- 计算着色器编译/加载
- SSBO管理
- FD/WLS梯度计算执行
- GPU计时（可选）

### 结构体
- RegularParams
  - dims[3]: 规则网格维度
  - origin[3]: 预留
  - spacing[3]: 预留
- WLSParams
  - wExponent: 权重指数
  - lambda: 正则项

### 核心变量
- shaderDir: 着色器目录
- progRegular/progWLS: FD/WLS程序对象
- ssbo0..ssbo4: 通用缓冲
- enableGpuTiming/timeQuery/lastGpuTimeMs: GPU计时状态

### 核心函数
- setShaderDir(dir): 设置着色器路径
- init(): 编译并链接着色器
- release(): 释放OpenGL资源
- computeRegularFD(positions,values,p,outGrad): FD计算（支持多分量）
- computeUnstructuredWLS(positions,offsets,neighbors,phi,p,outGrad): WLS计算
- setEnableGpuTiming(on): 开关GPU时间查询
- getLastGpuTimeMs(): 获取最近一次GPU纯计算时间

## 4. OpenGLManager.h（新增）

### 职责
- 独立OpenGL上下文管理
- 运行时信息记录（厂商/版本）
- 提供MakeCurrent能力

### 变量
- m_window: vtkOpenGLRenderWindow智能指针
- m_info: OpenGLRuntimeInfo
- m_ready: 是否初始化成功

### 函数
- initialize(offscreen): 创建并初始化上下文
- isReady(): 查询状态
- makeCurrent(): 设置当前上下文
- info(): 获取运行时信息
- window(): 获取底层窗口对象

---

## 5. CAEInterfaceTypes.h

### 职责
- 对外接口DTO（UI/测试/上层调用协议）
- 不承载内部几何与邻接细节

### 枚举
- CAEFieldAssociation: Point/Cell
- CAEGridClass: Regular/Unstructured
- CAEGradientMethod: Auto/FD/WLS

### 结构
- CAEFieldInfo: 字段基本信息
- CAEGradientRequest: 梯度计算请求
- CAEGradientResultMeta: 梯度结果元数据
- CAEDatasetSummary: 数据集摘要（用于UI左侧树）

---

## 6. CAEProcessingFacade.h/.cpp

### 职责
- 对外统一流程接口（读取/列举/计算/导出）
- 屏蔽内部转换与计算细节
- 供 TestGradient 和 Qt 层直接调用

### 私有结构
- DatasetRecord
  - id/displayName
  - data(DataObject)
  - sourceVtk
  - results(结果列表)

### 核心变量
- m_gl: OpenGL管理器
- m_engine: 梯度计算引擎
- m_initialized: 初始化状态
- m_records: 数据集仓库
- m_nextId: 自增ID

### 核心函数
- initialize(shaderDir): 初始化OpenGL与计算引擎
- loadDatasetFromVTKFile(path): 加载并转换，返回datasetId
- listDatasets(): 返回数据集摘要（含字段和结果）
- computeGradient(req,outMeta): 统一梯度计算入口
- exportDatasetToVTK(datasetId,outVtk): 导出用于渲染
- getArrayData(datasetId,arrayName,assoc,outData,outComps): 读取数组

### 私有辅助函数
- toGridClass / toDataArrayType / toAssociation: 类型映射
- fileNameFromPath: 路径到显示名
- makeResultName: 结果命名规则
- computeByFD: FD执行（支持规则网格点/单元）
- computeByWLS: WLS执行（支持点/单元）

---

## 7. TestGradient.cpp（建议定位）

### 定位
- 只做“接口调用 + 功能验证 + 对比输出”
- 不承载具体算法实现、OpenGL资源细节、数据转换细节

### 建议保留函数
- main:
  1) facade.initialize
  2) facade.loadDatasetFromVTKFile
  3) facade.listDatasets
  4) 遍历字段调用 computeGradient
  5) 输出结果名/状态

## 8. 后续Qt对接建议（与当前接口一致）
1. 读取按钮 -> loadDatasetFromVTKFile
2. 左侧树 -> listDatasets
3. 计算按钮 -> computeGradient
4. 结果渲染 -> exportDatasetToVTK + 选择结果数组渲染
5. 多文件多结果 -> datasetId隔离管理
