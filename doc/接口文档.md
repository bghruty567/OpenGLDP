# 接口文档

这份文档作为短版索引使用。

如果你要看当前有效接口，请优先看下面两份：

- `doc/api.md`
- `doc/系统接口文档.md`

## 1. 当前仍然有效的梯度接口

### 1.1 公开方法

- `CAEProcessingFacade::loadDatasetFromVTKFile(...)`
- `CAEProcessingFacade::listDatasets()`
- `CAEProcessingFacade::listFields(...)`
- `CAEProcessingFacade::computeGradient(...)`
- `CAEProcessingFacade::saveDatasetToVTKFile(...)`

### 1.2 当前有效的梯度方法

- `Auto`
- `FiniteDifference`
- `AdaptiveWeightedLeastSquares`

## 2. 当前公开执行路径

```text
RegularGrid      -> FD
UnstructuredGrid -> AWLS
```

## 3. 需要避免继续引用的旧内容

下面这些不应再作为“当前默认接口”写进论文：

- 旧的 `WeightedLeastSquares` 公开模式
- `SparseGradient.glsl`
- `SparseReconstruct.glsl`
- 以 `vtkCell::Derivatives(...)` 为默认主线的说法
- 按单元类型拆很多默认求导分支的说法
